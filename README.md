# Wallet - Billetera Virtual (Proyecto Frontend)

Este proyecto es una **billetera virtual** desarrollada como práctica de frontend.  
Permite simular operaciones básicas de una aplicación financiera, como:

- Visualizar el **saldo disponible**.
- Realizar **depósitos**.
- Enviar dinero a otros usuarios (**transferencias**).
- Revisar el **historial de transacciones**.

La aplicación está construida únicamente con **HTML, CSS, JavaScript y Bootstrap**, sin backend real.  
Los datos se manejan en el navegador (usando `localStorage`) para simular el comportamiento de una app bancaria.

## Estructura principal

- `index.html`: pantalla de inicio / login de la billetera.
- `menu.html`: menú principal con acceso a las diferentes secciones.
- `deposit.html`: formulario para realizar depósitos.
- `sendmoney.html`: formulario para enviar dinero.
- `transactions.html`: vista del historial de transacciones.

## Cómo usar el proyecto

1. Abre el archivo `index.html` en tu navegador (Chrome, Firefox, etc.) e inicia sesión con:
   - Correo: `admin@python.com`
   - Contraseña: `admin`
2. Navega por las diferentes páginas usando el menú de la aplicación.
3. Realiza pruebas de depósitos, envíos de dinero y revisa el historial de movimientos.

Este proyecto está pensado como ejercicio para practicar:

- Maquetación con **HTML**.
- Diseño responsivo y estilos con **CSS**.
- Manejo de eventos y lógica básica con **JavaScript**.
